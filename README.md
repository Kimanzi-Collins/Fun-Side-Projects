## what even is this?

Ever written a 10-line script at 2 AM just to see if something *works*? Yeah, that's this whole repo.

No frameworks. No architectures. No grand plan. Just **tiny, weird, sometimes useful code snippets** tossed into one place — like a junk drawer, but for code.

Some of it's Python. Some of it's JavaScript. Some of it might be Rust or Go or something I tried once at 3 AM and forgot about. The point is: **there are no rules here.**

## 🗂 what you'll find

| | |
|---|---|
| 🐍 **Python one-liners** | quick hacks, automations, curiosities |
| 🌐 **JS/TS tidbits** | DOM tricks, Node experiments, the usual chaos |
| 🦀 **Rust / Go / C** | when I feel brave (or reckless) |
| 🧪 **Random experiments** | things that probably shouldn't exist but do |
| 🎨 **Aesthetic scripts** | pretty terminal output, generative stuff |

> *If it made me go "huh, neat" — it's probably in here.*

---

## 🚀 how to use

```
1. Browse around
2. Find something cool
3. Copy it
4. Run it
5. Break it
6. Make it better (or worse, I don't judge)
```

There's no install process. No dependencies to wrestle with. Each snippet is self-contained. Grab what you want and go.
